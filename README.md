# Boxoffice-Dashboard-Project
YBIGTA 컨퍼런스

Lambda : AWS Lambda 에서 사용한 코드
- movie-crawling1,2 : 데이터 수집
- s3-to-elastic : AWS S3에 있는 데이터를 JSON 형식으로 변환하여 Elasticsearch로 데이터 보냄.

EC2 : EC2 환경에서 사용한 코드 
- Predict : 예상 누적 관객수 예측
- Word : 관객 반응 지수 및 워드클라우드
